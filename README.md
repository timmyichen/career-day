## Contributing

### Development Setup

Clone this repo, then install dependencies:

```
npm install
```

Make/copy your `.env` file:

```
MONGO_URI=
MONGO_TEST_URI=
```

Run the development server:

```
npm run dev
```

Run tests:

```
npm run test
```

Lint your files:

```
npm run lint
```

Find more scripts in `package.json`.
